# EduTrack 🚀

### Transforming Administration in Low-Resource Schools
EduTrack is a specialized MVP designed to bridge the digital divide in government schools in India. It replaces manual, paper-based registers with a high-performance, cloud-synced digital dashboard.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend-as-a-Service:** Supabase (PostgreSQL)
- **Deployment:** GitHub Pages

## ✨ Key Features
- **Real-time Persistence:** Attendance data is synced instantly to the cloud, preventing data loss common with paper records.
- **Role-Based Views:** Distinct interfaces for teachers and administrators to view classroom analytics.
- **Offline Readiness:** Designed to handle the low-connectivity environments of rural schools.

## 📈 Impact & Vision
Currently being field-tested to reduce teacher administrative overhead. The goal is to evolve this into an AI-driven tool that predicts student dropout risks based on attendance patterns.
